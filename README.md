# Rust Hangman

A simple terminal Hangman game implemented in Rust.

## Building
Requirements:
- Rust 1.75+ (`rustc` and `cargo`)

Clone the repo:
```bash
git clone https://github.com/WrongNicholas/rust-hangman.git
cd rust-hangman
```

Run in development mode:
```bash
cargo run
```

## Installing
Install locally as a binary:
```bash
cargo install --path .
```

Then run:
```bash
hangman
```
